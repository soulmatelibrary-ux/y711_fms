Y711 FMS 사용 가이드

시스템 소개

Y711 FMS는 서해안 항공로(Y711)를 이용하여 제주(RKPC)로 향하는 항공편의 CTOT(Calculated Take-Off Time, 계산된 이륙시간)를 관리하는 시스템입니다.

여러 공항에서 출발한 항공기들이 동일한 항공로에서 합류할 때 안전한 분리 간격을 유지하도록 출발 시간을 자동으로 조정합니다.

대상 공항 및 합류 지점

공항 코드 합류지점 진입시간
김포 RKSS BULTI 8분
청주 RKTU MEKIL 7분
군산 RKJK MANGI 3분
광주 RKJJ DALSU 1분
항공로 경로:
BULTI → MEKIL → GONAX → BEDES → ELPOS → MANGI → DALSU → NULDI → DOTOL → 제주

CTOT 계산 원리

1단계: EOBT 기준

CTOT는 EOBT(예상 이륙 시간)보다 빠를 수 없습니다.

2단계: 이륙 간격 적용

같은 공항에서 연속 출발하는 항공기는 최소 이륙 간격을 유지합니다.

3단계: 웨이포인트 분리

합류 지점에서 앞 항공기와 최소 3분 분리를 유지하도록 CTOT를 조정합니다.

핵심 원칙: 각 웨이포인트에서 항공기 간 충돌(3분 미만 분리)이 발생하지 않도록 뒤따르는 항공기의 CTOT를 자동으로 지연시킵니다.

화면 구성

좌측 패널 - 항공편 목록
  • 날짜/요일 선택으로 스케줄 필터링
  • 항공편 드래그앤드롭으로 순서 변경 가능
  • EOBT, CTOT 직접 수정 가능
  • 항공편 클릭 시 기준 항공기로 설정 (빨간색 표시)
우측 상단 - 타임라인
  • 공항별로 항공편 시간대 시각화
  • 노란색: CTOT가 지연된 항공기
  • 빨간색 경고: 충돌 위험
우측 하단 - Live Route Map
  • Full View 버튼으로 전체 화면 전환
  • 재생 버튼으로 시뮬레이션 시작
  • 항공기가 각 합류 지점까지 상승 후 순항
  • 분리 간격 실시간 표시

사용 방법

1. 데이터 로드

상단의 Excel 버튼으로 스케줄 파일 업로드
(CALLSIGN, DEPT, DEST, CFL, EOBT_UTC, DAY_OF_WEEK 컬럼 필요)

2. CTOT 계산

Refresh CTOT 버튼 클릭 또는 항공편 순서 변경 시 자동 계산

3. 시뮬레이션

▶ 재생 버튼으로 시뮬레이션 시작
속도 조절: 1x, 2x, 5x, 10x, 20x

4. 설정 변경

Settings에서 진입 시간, 웨이포인트 구간 시간, 분리 기준 조정 가능
